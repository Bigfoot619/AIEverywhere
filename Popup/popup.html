AIEverywhere Features

  • Improve English: Enhances the English of selected text as if an English teacher wrote it.
  • Improve English - Creative: Similar to "Improve English" but with settings adjusted for higher creativity.
  • Add Comments to Code: Adds explanatory comments to selected code snippets.
  • Summarize to a Single Paragraph: Condenses the selected text into a single, comprehensive paragraph.
  • AI Quiz: Generates ten multiple-choice questions based on the selected text. Correct answers are highlighted in green and bold.
